APP KASIR ☕

  

Aplikasi kasir berbasis Laravel untuk mengelola transaksi di kafe dengan tampilan modern dan fitur lengkap.

🚀 Fitur Utama

✅ Manajemen Menu - Tambah, edit, dan hapus menu kafe dengan mudah.✅ Transaksi Cepat - Proses transaksi dengan detail pembayaran yang akurat.✅ Laporan Keuangan - Pantau pendapatan harian dan bulanan dalam grafik interaktif.✅ Filter Data - Cari transaksi berdasarkan kasir atau rentang tanggal.✅ Riwayat Aktivitas - Pantau aktivitas pengguna dengan log sistem.

🛠 Teknologi yang Digunakan

Backend: Laravel 8.x

Database: MySQL 5.x

Frontend: TailwindCSS, Bootstrap

Ikon: FontAwesome, Lucide Icons

🎨 Tampilan

tunggu besok

📜 Lisensi

Aplikasi ini dibuat untuk keperluan internal kafe. Anda bebas menggunakan dan memodifikasi sesuai kebutuhan.
